¿Qué es HTML?

HTML es un lenguaje de marcado que se usa para estructurar y presentar contenido en la web.

Estructura básica de una página HTML

Una página HTML se compone de etiquetas como <html>, <head> y <body>, que organizan el contenido.

CSS: Estilizando HTML

CSS es un lenguaje de estilos que permite dar formato a las páginas HTML, cambiando colores, tamaños, márgenes, y más.

Selectores en CSS

Los selectores en CSS permiten seleccionar elementos específicos de HTML para aplicarles estilos.
